# Splint

A **S**WI-**P**ro**l**og **int**erface library for Rust, intended for Rust applications that embed SWI-Prolog as a scripting system.

This library is heavily vibe-coded, guaranteed to be unstable, and fully subservient to the needs of one of my other projects. It wouldn't be a stretch to say that it's mostly public for the free CI minutes.
